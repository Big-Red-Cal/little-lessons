# Guide

In pairs, one person receives a pile of CISC style instructions, and the other a pile of RISC style instructions.

Each person also receives a blank register grid to return their values.

Each cell in the grid will only hold one, character/digit.

Learners will place their instructions face down, take an instruction from the top of the pile, execute it, then repeat the next instruction.
# CISC

In reg_1, return "500"

In reg_2, return '1'

In reg_7, return  (reg_2 * 10) / ((((100 / 2) / 5) * reg_1)

In reg_8, return "All_of_this_ridi"

In reg_9, return "culous_maths_jus"

In reg_10, return "t_to_get_a_"

In reg_10, append reg 7

Hold up an arm and read the values of reg_8, reg_9, reg_10 + "this was too complex"

# RISC

In reg_1, return '5'

In reg_1, append '0'

In reg_1, append '0'

In reg_2, return '1'

In reg_3, return reg_2 * 10

In reg_4, return 100 / 2

In reg_5, return reg_4 / 5

In reg_6, return reg_5 * reg_1

In reg_7, return reg_3 / reg_6

In reg_8, return "All_"

In reg_8, append "of_"

In reg_8, append "this_"

In reg_8, append "ridi"

In reg_9, return "culous_"

In reg_9, append "maths_"

In reg_9, append "jus"

In reg_10, return "t_"

In reg_10, append "to_"

In reg_10, append "get_"

In reg_10, append "a_"

In reg_10, append reg_7

Hold up an arm

Read the values of reg_8, reg_9, reg_10 + "this was too simple"
